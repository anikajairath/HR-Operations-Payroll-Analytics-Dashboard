# 📊 HR Operations & Payroll Analytics Dashboard

## Overview

An end-to-end workforce analytics project analyzing employee, payroll, expense claim, and travel expenditure data using Python and Power BI.

The project focuses on workforce cost monitoring, payroll analysis, reimbursement tracking, and travel expenditure insights through interactive dashboards and business intelligence reporting.

## Tools Used

* Python (Pandas)
* Power BI
* DAX
* Excel

## Key Features

* Data Cleaning & Validation
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Data Integration using EmployeeID
* Interactive Power BI Dashboards
* Workforce, Payroll, Expense & Travel Analytics

## Dashboard Modules

### Payroll Analytics

* Total Employees
* Gross & Net Payroll
* Department-wise Payroll Analysis
* Salary Trends

### Expense Analytics

* Claim Analysis
* Approval Rate Tracking
* Expense Category Insights

### Travel Analytics

* Travel Spend Analysis
* Destination Trends
* Travel Mode Analysis

## Key Insights

* Payroll is a major operational cost center.
* Travel, Mobile, and Hotel claims represent significant expense categories.
* Flight travel contributes the highest travel expenditure.
* Workforce and spending trends can support budgeting and operational planning.
